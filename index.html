Module Immutable Persistent Data Structures

Open console to play with ImmutableList, ImmutableVector, ImmutableMap, and ImmutableSet.

Example

        > var a = new ImmutableList(0, 1, 2, 3);
        > var b = a.push(4, 5, 6);
        > a.toArray(); // -> [0, 1, 2, 3]
        > b.toArray(); // -> [0, 1, 2, 3, 4, 5, 6]
